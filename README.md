# Muharram Programming Language

An interpreted, object-oriented programming language with support for classes, properties, and variables.

## Overview

Muharram is a dynamically-typed interpreted language designed with simplicity and OOP principles in mind. It supports:
- **Classes and inheritance**
- **Properties and methods**
- **Variables**
- **Dynamic typing**

## Project Structure

- `interpreter/` - Core interpreter implementation
- `lexer/` - Tokenization logic
- `parser/` - AST generation
- `examples/` - Sample Muharram programs
- `docs/` - Language documentation and syntax guide

See `docs/` for language syntax and examples.
